# Amazon CloudWatch API

Amazon CloudWatch is a monitoring and observability service that provides data and actionable insights to monitor applications, respond to system-wide performance changes, and optimize resource utilization, offering a unified view of operational health across AWS resources, applications, and services.

## APIs

- **Amazon CloudWatch API** - Core API for monitoring AWS resources and applications in real time, collecting and tracking metrics, creating alarms, managing dashboards, and automatically reacting to changes in your AWS environment.
