Amazon CloudWatch Application Signals now supports runtime metrics for Java, Python and .Net applications. This new capability lets you monitor application runtime metrics directly in Amazon CloudWatch, helping you correlate application performance signals and improve application observability. Amazon CloudWatch Application Signals automatically instruments your applications running on AWS, so that you can monitor your application health, troubleshoot performance, and track against your business objectives.
